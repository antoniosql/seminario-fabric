  # **Crear la capacidad de Fabric**
 
 
 - Desde el portal de Azure, mostrar la creación de una capacidad de Fabric
 - Mostrar las opciones y propiedades --> Pausar y Reiniciar
	 - [Tutorial - Start and Stop Microsoft Fabric Capacity - Lytix](https://lytix.be/tutorial-start-and-stop-microsoft-fabric-capacity/) 
	 - [Pause and resume your capacity - Microsoft Fabric | Microsoft Learn](https://learn.microsoft.com/en-us/fabric/enterprise/pause-resume) 
 - Comentar la diferencia entre darle permisos al recurso de Fabric, y asignar el rol de "Administrador Fabric" desde los Roles del portal de M365
# **Asignar la Capacidad**

- Crear una nuevo espacio de trabajo y asignar la capacidad
- Ir al centro de administración y mostrar las capacidades
- Mostrar las configuraciones de Fabric en la configuración general
